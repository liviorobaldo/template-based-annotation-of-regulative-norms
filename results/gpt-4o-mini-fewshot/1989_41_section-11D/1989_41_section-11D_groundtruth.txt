IT IS PROHIBITED
FOR a child arrangements order
TO impose an activity condition on an individual who is a child
UNLESS the individual is a parent of the child concerned. [source: https://www.legislation.gov.uk/ukpga/1989/41/section/11D]
------------------------
IT IS PROHIBITED
FOR a child arrangements order
TO impose an activity condition
WHEN/IF/WHERE it is an excepted order (within the meaning given by section 11B(4)). [source: https://www.legislation.gov.uk/ukpga/1989/41/section/11D]
------------------------
IT IS PROHIBITED
FOR a child arrangements order
TO be varied so as to impose an activity condition
WHEN/IF/WHERE it is an excepted order (within the meaning given by section 11B(4)). [source: https://www.legislation.gov.uk/ukpga/1989/41/section/11D]
------------------------
IT IS PROHIBITED
FOR a child arrangements order
TO impose an activity condition on an individual
UNLESS the individual is habitually resident in England and Wales. [source: https://www.legislation.gov.uk/ukpga/1989/41/section/11D]
