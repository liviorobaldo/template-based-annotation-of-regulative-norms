IT IS PROHIBITED
FOR child arrangements order
TO impose an activity condition on an individual who is a child
UNLESS the individual is a parent of the child concerned
------------------------
IT IS PROHIBITED
FOR child arrangements order
TO impose an activity condition
IF the order is an excepted order (within the meaning given by section 11B(4)) [source: section 11B(4)]
------------------------
IT IS PROHIBITED
FOR child arrangements order
TO be varied so as to impose an activity condition
IF the order is an excepted order (within the meaning given by section 11B(4)) [source: section 11B(4)]
------------------------
IT IS PROHIBITED
FOR child arrangements order
TO impose an activity condition on an individual
UNLESS the individual is habitually resident in England and Wales
------------------------
IT IS PROHIBITED
FOR activity condition
TO continue to have effect
IF the individual subject to the condition ceases to be habitually resident in England and Wales
